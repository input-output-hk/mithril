macro_rules! register_config_value {
    ( $map:ident, $namespace:expr, $self:ident.$command:ident ) => { ... };
    ( $map:ident, $namespace:expr, $self:ident.$command:ident, $mapping:expr ) => { ... };
    ( $map:ident, $namespace:expr, $command:ident = $value:expr ) => { ... };
}
Expand description

Register a parameter in the config map using the identifier as key. Example: register_config_value!(map, namespace, self.identifier)

The same macro, with a different syntax, is used to insert the given value without transformation. Iit is designed to be used by other macros. Example: register_config_value!(map, namespace, identifier = value)
